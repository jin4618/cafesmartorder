### 4.핵심기능

>Client ( 회원가입 및 탈퇴, 장바구니, 주문, 포인트 적립, 인기메뉴 확인 )   
- 메뉴를 조회할 수 있다.
- 원하는 메뉴를 담을 수 있다.
- 담은 메뉴들을 결제할 수 있다.
- 포인트를 적립하고 5개가 모일 시 아메리카노 무료 쿠폰을 받을 수 있다.
- 매출 기반 인기 메뉴를 확인할 수 있다.
- 회원 가입 및 탈퇴를 할 수 있다.
